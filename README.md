#GhostEroid
This is basically a website I made with html css and js, in which i created a simple game ( in future may more will come ). Kept it scary themed to bring change to the projects.
#Languages
HTML: skeleton of the website, adding divs for different parts
CSS: for styling and giving the horror theme and fonts, by adding a font form the internet and scary background images
JS: made the app function with logic
#Reason
I wanted to try a new themed website as I became familiar with making websites ( it was easier to make one now ).
#Features
beautiful transition to the realm once you keep your cursor over the girls mouth. 4 stages of game. option to return to the homepage once game is finished. gives chills
